## Installing uv
```bash
pip install uv
```

Install Python through uv
```
uv python install 3.12
```

Create a project using
```bash
uv init
```

Install packages using uv
```bash
uv add scikit-learn pytorch pandas numpy matplotlib jupyterlab ipykernel
```

To open JupyterLab:
```bash
uv run jupyter lab
```
